# CCIRS Frontend (React + Tailwind)

## Run

```bash
cd frontend
npm install
npm run dev
```

## Stack

- React (functional components + hooks)
- React Router
- Context API
- Tailwind CSS
- Lucide Icons

## Structure

```txt
src/
  components/
  pages/
  services/
  context/
  hooks/
  utils/
  assets/
  App.jsx
```
